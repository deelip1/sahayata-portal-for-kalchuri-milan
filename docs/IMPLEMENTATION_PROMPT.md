# Sahayata Portal – Final Build Prompt (Revised)

Build a standalone Sahayata portal on subdomain with a modern Bootstrap 5.3+ UI, AI-powered admin and frontend experience, and modular Laravel architecture.

## Must-have modules
1. **Admin Dashboard**: active cases, contributions, pending approvals, team/member widgets, AI insights panel.
2. **Team Management**: full CRUD for roles, photos, contact visibility, district, bio, status.
3. **Member Management**: full CRUD for member records with searchable/filterable listing.
4. **Frontend Team Directory**: card-based leadership/district/volunteer sections with smart search.

## AI UX requirements
- Smart form suggestions
- Validation assistance + correction hints
- Duplicate/fraud risk hinting hooks
- Gentle reminder workflow hooks
- Clean language guidance for rural usability

## Current repository delivery
- Controllers/models for dashboard, team, and member modules.
- Migrations for `sahayata_team` and `sahayata_members`.
- Bootstrap-based views for frontend and admin sections.
- Shared AI helper JavaScript across forms and filters.
